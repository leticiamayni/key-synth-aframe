# key-synth-aframe
<h2>Simple visual 3d synth using Tone.js and A-Frame.</h2>

<h3>Use the keys: "H", "J", "K" and "L" on keyboard to interact with the sound cubes.</h3>

<p>Library Versions</p>
<p>A-Frame 0.8.0</p>
<p>Tone.js 13.8.25</p>
